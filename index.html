How to use this chat application:

  1. Start the Backend: First, make sure your Java Spring Boot backend is running on port 8080
  2. Open Chat: Click the "Open Chat App" button above to start chatting
  3. Enter Username: Enter your username when prompted
  4. Start Chatting: Send messages and see them appear in real-time!

To start the backend:

  1. Open a terminal in the backend directory
  2. Run: mvn spring-boot:run
  3. Wait for "Started ChatApplication" message
  4. Then click "Open Chat App" above
